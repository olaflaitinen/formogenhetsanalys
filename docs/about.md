# About Förmögenhetsanalys

Förmögenhetsanalys is a research package for estimating wealth concentration in Sweden using heterogeneous graph neural networks.

## Purpose

The package aims to provide reproducible, graph-aware methods for wealth distribution estimation that:
- Combine register data with ownership network analysis
- Apply state-of-the-art GNN architectures to infer hidden ownership
- Provide rigorous uncertainty quantification through Bayesian methods
- Align micro-data with macro totals via Distributional National Accounts

## Research Context

Wealth inequality in Sweden is traditionally measured using survey data and register data separately. This package integrates both by modeling the ownership network that connects households to firms and assets, enabling more accurate estimation of top wealth shares and inequality indices.
