# GDPR Compliance

This document describes how Förmögenhetsanalys complies with the General Data Protection Regulation (GDPR).

## Legal Basis

### Article 6(1)(e): Public Interest
- Processing wealth register data for statistical research
- Swedish public interest in understanding wealth distribution
- Authorized under Swedish Statistics Act (2001:99)

### Article 9(2)(j): Scientific Research
- Processing special category data (wealth) for research
- Ethics committee approval required
- Appropriate safeguards in place

## Data Minimization

- Only necessary wealth components are collected
- Data is aggregated where possible
- No personal identifiers in published results

## Purpose Limitation

- Data is used solely for wealth concentration estimation
- No commercial use
- No use for individual decision-making

## Data Security

- Processing in secure SCB environment
- Encryption in transit and at rest
- Access controls and logging
- Regular security audits

## Data Subject Rights

### Right to be Informed
- Participants informed through SCB data collection
- This documentation explains processing purposes

### Right to Access
- Data subjects can request access via SCB
- Direct access through package not available

### Right to Rectification
- Corrections handled by source authorities (Skatteverket)

### Right to Erasure
- Not applicable for register data held by public authority
- Retention period governed by Swedish archival law

### Right to Restrict Processing
- Can be requested via SCB
- May affect research validity

## Data Transfers

- No cross-border transfers
- All processing within EU/EEA
- Sweden is GDPR-compliant jurisdiction

## Data Protection Impact Assessment (DPIA)

A DPIA has been conducted for this research project. See `dpia-summary.md` for details.

## Data Retention

- Research data retained according to SCB policies
- Synthetic data (CC0-1.0) retained indefinitely
- Published results do not contain personal data
