# Swedish Legal Basis

This document describes the Swedish legal framework for wealth register data access and use.

## Legal Foundations

### Skatteförvaltningslagen (2011:1244)
Governs the Swedish Tax Agency's handling of tax data, including wealth register data.

### Skattebetalningslag (1997:483)
Regulates tax data collection and confidentiality.

### Offentlighets- och sekretesslag (2009:400)
- Public Access to Information and Secrecy Act
- Wealth register data is classified as sensitive personal data
- Requires special handling and confidentiality

## Data Access

### SCB (Statistics Sweden) Microdata Access
- Requires research application
- Ethics committee approval required
- Data access agreement mandatory
- Data must be processed in secure environment

### SCB Microdata Portal
- Online access to aggregated statistics
- No individual-level data available
- Publicly available under open data license

## Data Protection

### GDPR Compliance
- Wealth register data contains personal data
- Requires lawful basis for processing (research with public interest)
- Data minimization principle applies
- Right to be informed for data subjects

### Secrecy (Sekretess)
- Individual wealth data is classified as confidential
- Disclosure is criminal offense (Brottsbalken 20 kap. 3 §)
- Research results must be aggregated to protect confidentiality

## Ethical Approval

### Swedish Ethical Review Authority (Etikprövningsmyndigheten)
- Required for research using personal data
- Application must describe purpose, methods, and data handling
- Approval is mandatory before data access

## Swedish Data Protection Authority (IMY)
- Supervises GDPR compliance
- Can impose fines for violations
- Requires breach notification within 72 hours
