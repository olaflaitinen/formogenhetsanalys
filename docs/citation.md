# Citation

This document provides citation information for Förmögenhetsanalys.

## How to Cite

### BibTeX
```bibtex
@software{formogenhetsanalys2025,
  author = {Laitinen-Fredriksson Lundström Imanov, Gustav Olaf Yunus},
  title = {Förmögenhetsanalys: Graph-Aware Wealth Concentration Estimation},
  year = {2025},
  url = {https://github.com/olaflaitinen/formogenhetsanalys},
  version = {0.1.0},
  license = {EUPL-1.2}
}
```

### APA
Laitinen-Fredriksson Lundström Imanov, G. O. Y. (2025). Förmögenhetsanalys: Graph-aware wealth concentration estimation (Version 0.1.0) [Computer software]. https://github.com/olaflaitinen/formogenhetsanalys

### MLA
Laitinen-Fredriksson Lundström Imanov, Gustav Olaf Yunus. "Förmögenhetsanalys: Graph-Aware Wealth Concentration Estimation." 2025. GitHub, https://github.com/olaflaitinen/formogenhetsanalys.
