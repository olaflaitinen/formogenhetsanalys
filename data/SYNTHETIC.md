# Synthetic Data

This directory contains synthetic data for testing and demonstration purposes.

## Purpose

Synthetic data is used to:
- Test the package without access to real register data
- Demonstrate functionality in examples and documentation
- Provide reproducible benchmarks
- Enable open-source development without GDPR restrictions

## Files

### households.parquet
Synthetic household wealth data generated by `synthetic_wealth_register()`.

**Columns:**
- household_id
- total_wealth
- financial_wealth
- real_estate_wealth
- business_wealth
- debt
- year

**Generation:**
```python
from formogenhetsanalys.ingestion.wealth_register import synthetic_wealth_register
df = synthetic_wealth_register(n=200, seed=19960307)
df.write_parquet("data/synthetic/households.parquet")
```

### firms.parquet
Synthetic firm data generated by `synthetic_firm_register()`.

**Columns:**
- firm_id
- org_nr
- is_fåmansföretag
- equity_book
- revenue
- profit
- sector_code
- year

### assets.parquet
Synthetic asset data.

**Columns:**
- asset_id
- asset_type
- value_market
- value_book
- isin
- year

### edges_ownership.parquet
Synthetic ownership edges.

**Columns:**
- source_id
- target_id
- ownership_share
- ownership_type
- year

### edges_kinship.parquet
Synthetic kinship edges.

**Columns:**
- source_id
- target_id
- kinship_type
- year

### graph.graphml
Synthetic wealth network in GraphML format.

## Regeneration

To regenerate synthetic data:
```bash
python scripts/make_synthetic_graph.py
```

This will regenerate all synthetic data files with the default seed (19960307).

## Privacy

This synthetic data contains no real household or firm information. All entities are fictional and any resemblance to real entities is coincidental.
